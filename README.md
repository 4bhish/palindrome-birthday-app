# Build Palindrome Birthday App.
A web app to know if your  birthday is palindrome.

## How to use:
- Enter your Birth Date.
- You will get know if your birthday is palindrome or not.
- also you will get to know next palindrome date.

## Tech used:
- HTML5
- CSS
- JavaScript
